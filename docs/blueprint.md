# **App Name**: Am I Cute?

## Core Features:

- Interactive Questions: Presents a series of questions on successive pages to the user. Navigates smoothly to the next question upon confirmation, engaging users with interactive elements and providing a consistent, user-friendly flow.
- Evasive 'No' Button: Implement an evasive 'No' button that dynamically repositions itself to avoid cursor interaction, compelling users to affirm each question positively, achieved with javascript event listeners
- Click Counter: Track the number of 'Yes' button clicks on the final gate page, unlocking a surprise message upon reaching the required count, enhancing user engagement through a rewarding, interactive element.
- Surprise Message Display: Show a personalized affirmation message with animated hearts and smooth transitions to delight the user, fostering a positive and uplifting user experience.
- Animated Hearts Effect: Adds a gentle animated floating hearts visual effect, enhancing the charm and visual appeal.

## Style Guidelines:

- Background color: Very light pink (#F8E8EE) for a soft, comforting backdrop.
- Primary color: Pale lavender (#E6E6FA) to create a light and playful mood.
- Accent color: Soft coral (#F08080) to add a subtle touch of warmth and fun, and for contrast with primary and background
- Font: 'Poppins' (sans-serif) for a modern, friendly feel, suitable for both headlines and body.
- Use a full-screen, centered layout to maintain focus and readability across devices.
- Incorporate smooth transitions and fade-in effects to enhance the user experience, using CSS animations or Javascript to subtly animate changes between the successive pages of questions and, especially, in the final surprise section.
- Employ floating hearts animation for added visual appeal, enhancing the heartwarming design of the app.
